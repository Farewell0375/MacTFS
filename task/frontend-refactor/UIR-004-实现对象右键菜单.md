## 目标

将文件、文件夹、目录树节点、目录对比结果项、Pending Changes 项的对象操作统一迁移到右键菜单。

## 实现范围

- 左侧目录树节点接入右键菜单。
- 中间文件列表文件夹接入右键菜单。
- 中间文件列表文件接入右键菜单。
- Pending Changes Included / Excluded 项接入右键菜单。
- 目录对比结果项接入右键菜单。
- 取消中间文件列表选中项底部操作区。
- 文件夹菜单包含：打开、获取最新版本、签出、新增本地文件到服务器、删除、撤销挂起更改、查看历史、与服务器目录对比、映射到本地。
- 文件菜单包含：查看、获取最新版本、签出、删除、撤销挂起更改、查看历史、比较本地与服务器 Latest。
- Pending Changes 文件项包含：查看、比较本地与服务器 Latest、撤销、查看历史、移到 Included、移到 Excluded。
- Pending Changes 文件夹项不展示内容比较。
- pending add 文件只显示查看和撤销，不显示比较本地与服务器 Latest。
- Checkin 不放到单个文件或文件夹右键菜单。
- 根据 mapped、pending edit、pending add、folder、localPath 等状态控制菜单可用性。

## 不在范围

- 不实现具体弹窗内容。
- 不实现键盘快捷键。
- 不实现 Checkin 弹窗。
- 不实现手动冲突块编辑。

## 验收标准

- 左侧树、文件列表、Pending Changes 均可打开右键菜单。
- 顶部和底部不再承担文件对象操作。
- pending edit 文件禁止直接 Get Latest。
- pending add 文件不显示服务器比较。
- 文件夹与服务器目录对比入口可用。
- 文件比较本地与服务器 Latest 入口可用。
- Checkin 仍只在右侧 Changes 面板提交 Included Changes。
