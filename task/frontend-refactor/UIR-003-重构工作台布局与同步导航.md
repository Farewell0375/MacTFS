## 目标

重构主工作台布局，让左侧 Source Tree、右侧 Changes、底部操作日志都支持收起，并保证左侧树和中间文件列表导航状态一致。

## 实现范围

- `AppShell` 删除顶部文件/目录操作按钮。
- 顶部只保留连接状态、服务器、Collection、Workspace、刷新和设置等全局入口。
- 左侧 Source Tree 支持展开/收起。
- 左侧收起后只显示展开按钮，不显示路径或图标列表。
- 右侧 Changes 支持展开/收起。
- 右侧收起后只显示展开按钮，不显示 Pending Changes 数量徽标。
- 底部操作日志支持展开/收起。
- 不实现拖拽调整宽度。
- 当前路径和左侧目录树展开路径统一提升到 `home.tsx`。
- 用户在中间双击或右键“打开”进入目录时，左侧树自动展开并选中对应节点。
- 用户在左侧树选择目录时，中间文件列表切换到同一路径。
- 中间文件列表单击只选中，不进入目录。
- 第一版不支持键盘快捷键。

## 不在范围

- 不做拖拽调整宽度。
- 不做自定义布局保存。
- 不做键盘快捷键。
- 不做文件查看、Diff、History 弹窗。
