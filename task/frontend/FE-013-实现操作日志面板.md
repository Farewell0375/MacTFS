## 目标

实现底部操作日志面板，帮助判断网络卡顿、TFS 异常和操作结果。

## 实现范围

- 展示操作名称
- 展示开始时间、结束时间、耗时
- 展示路径摘要
- 展示成功 / 失败
- 展示错误信息
- API 请求期间显示操作中状态

## 不在范围

- 不做日志导出
- 不做实时流式日志
- 不做异步 Job

- TFS 网络慢时 UI 明确显示正在执行
- 失败信息可读
