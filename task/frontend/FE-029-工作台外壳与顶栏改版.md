## 目标

把三栏工作台外壳与顶栏升级为 macOS 工具栏 / Finder 侧栏风格，并补齐布局动效。

## 实现范围

- 顶栏按 macOS 工具栏风格重排：为红绿灯预留左侧空间，上下文信息居左、动作按钮图标化居右
- 左侧目录树面板改 Finder 式侧栏：毛玻璃背景（接 FE-027 vibrancy）、圆角选中条、分组标题样式
- 左右面板折叠/展开增加宽度过渡动画，底部操作台增加滑入滑出动画
- 统一三栏分隔线、内边距与留白

## 不在范围

- 目录树节点与文件列表行内样式（FE-030 做）
- 弹窗样式（FE-031 做）
- 不改动作编排与状态模型

## 涉及文件

- `mactfsui/app/components/app/workspace-shell.tsx`
- `mactfsui/app/components/app/top-bar.tsx`

## 验收标准

- 顶栏与红绿灯融合自然，动作按钮可用且有 tooltip
- 侧栏呈毛玻璃 Finder 风格
- 折叠/展开、底部操作台均有流畅动画且不卡顿
- 同步导航、刷新等既有行为无回归

## 测试方式

```bash
cd mactfsui && pnpm typecheck && pnpm electron:dev
# 手工验证：折叠/展开左右面板与底部操作台，检查动画与功能
```
