## 目标

实现右侧 Pending Changes 面板，并支持 Included / Excluded 分组。

## 实现范围

- 查询 pending changes
- 展示 Included Changes
- 展示 Excluded Changes
- 文件可在两组之间移动
- 状态只保存在当前 UI 会话

## 不在范围

- 不持久化 Included / Excluded
- 不和左侧 / 中间选中文件联动
- 不做 Work Item

- 用户可移动到 Excluded
- 签入只读取 Included 数据
