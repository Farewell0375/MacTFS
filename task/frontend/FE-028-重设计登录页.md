## 目标

把登录页从单张表单卡片升级为有品牌感、有动效、元素丰富的现代登录页。

## 实现范围

- 左右分栏布局：左侧品牌区（logo、产品名、标语、版本号、氛围背景动效），右侧连接表单卡片
- 表单 → Collection 选择两阶段切换增加滑动 + 淡入淡出过渡
- 新增：密码可见性切换按钮、最近连接记录（localStorage 多条、一键填充）、本地服务健康状态指示灯
- 连接中按钮内 spinner、阶段文案过渡；连接失败表单抖动 + 错误提示条
- Collection 列表改卡片式：悬停浮起、选中高亮过渡
- 保持现有 `api.connect / listCollections / ensureWorkspace` 调用链路不变

## 不在范围

- 不改连接协议与认证方式
- 不改 `app/lib/api` 接口契约（最近连接仅存前端本地）

## 涉及文件

- `mactfsui/app/components/app/connect-view.tsx`
- 新增 `mactfsui/app/lib/recent-servers.ts`（或等价的本地存储工具）

## 验收标准

- 登录页具备品牌区与氛围背景，观感现代精致
- 两阶段切换有流畅过渡动效
- 最近连接可保存、可一键填充；密码可见性切换可用
- 连接成功进入工作台的链路与改版前完全一致

## 测试方式

```bash
cd mactfsui && pnpm typecheck && pnpm electron:dev
# 手工验证：连接真实 TFS → 选择 Collection → 进入工作台
```
