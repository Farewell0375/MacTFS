## 目标

实现 checkout、add、delete、undo、get latest 的 UI 操作入口。

## 实现范围

- 已下载文件可 checkout
- 已下载目录可递归 checkout 并二次确认
- localOnly 可 add
- 已版本控制文件可 pending delete
- pending change 可 undo
- notDownloaded 可 Get Latest

## 不在范围

- 不做自动 checkout
- 不做未下载服务端文件删除
- 不做冲突解决器

- 目录 checkout 前显示影响数量确认
- 不同状态展示正确操作
- 操作后刷新相关列表或提示用户重新对比
