## 目标

把左侧目录树与中间文件列表的行样式、图标、加载与空状态升级到精致水平。

## 实现范围

- 树节点与文件行统一行高、圆角悬停/选中态；展开箭头旋转动画、子级展开渐入
- 按扩展名区分文件图标与颜色，文件夹/分支/已映射目录差异化图标
- 列表加载改骨架屏；空目录、未连接等空状态配图标 + 引导文案
- pendingEdit / pendingAdd / pendingDelete / 未下载等状态徽标统一配色与形状
- 文件列表行级联渐入动效（首次加载时）

## 不在范围

- 不改目录加载、右键菜单与动作编排逻辑
- 不改服务端接口与数据结构

## 涉及文件

- `mactfsui/app/components/explorer/source-tree-panel.tsx`
- `mactfsui/app/components/explorer/folder-items-panel.tsx`

## 验收标准

- 树与列表行有清晰的悬停/选中反馈，展开收起有动画
- 不同类型文件图标可区分，状态徽标风格统一
- 加载呈骨架屏，空状态有引导
- 导航、右键菜单、双击打开等既有行为无回归

## 测试方式

```bash
cd mactfsui && pnpm typecheck && pnpm electron:dev
# 手工验证：浏览多层目录、含挂起更改的目录，检查图标与徽标
```
