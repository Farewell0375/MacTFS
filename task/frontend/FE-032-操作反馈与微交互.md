## 目标

补齐全局操作反馈（toast、进度）与控件微交互，让操作结果即时可感知。

## 实现范围

- 引入轻量 toast 通知（成功/失败/进行中），接入既有动作编排的结果回调，与日志面板互补
- 挂起更改面板、日志面板按新 token 改版；新日志条目滑入动画
- 控件微交互：按钮按压缩放、输入框聚焦光环、复选框勾选动画、hover 过渡
- 长操作（GetLatest、Checkin、目录对比）期间展示进度反馈动效

## 不在范围

- 不改动作编排逻辑与日志数据结构
- 不做系统级通知（macOS 通知中心）

## 涉及文件

- `mactfsui/app/components/inspector/changes-panel.tsx`
- `mactfsui/app/components/logs/console-panel.tsx`
- `mactfsui/app/components/ui/button.tsx`、`input.tsx`、`checkbox.tsx`
- 新增 toast 组件与挂载点（`app/root.tsx` 或 workspace-shell）

## 验收标准

- 关键操作（签出、签入、获取、撤销等）成功/失败均有 toast 反馈
- 面板样式与全局风格统一，日志新条目有动画
- 控件具备统一微交互，无性能问题
- 既有操作链路无回归

## 测试方式

```bash
cd mactfsui && pnpm typecheck && pnpm electron:dev
# 手工验证：执行签出/撤销/获取最新，观察 toast 与面板反馈
```
