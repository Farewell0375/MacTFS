## 目标

实现明暗主题自动跟随 macOS 系统外观，并提供手动切换入口。

## 实现范围

- 监听系统外观（`prefers-color-scheme` / Electron nativeTheme），自动切换 `.dark`
- 顶栏增加三态切换：跟随系统 / 亮色 / 暗色，选择持久化到 localStorage
- 主题切换时全局颜色过渡动画
- 校验全部页面、面板、弹窗、Monaco Diff 在 dark 下的对比度与可读性，修正不达标 token

## 不在范围

- 自定义主题色 / 多主题方案
- Monaco 编辑器自定义配色（仅在明暗两套内置主题间切换）

## 涉及文件

- `mactfsui/app/app.css`
- `mactfsui/app/components/app/top-bar.tsx`
- 新增 `mactfsui/app/hooks/use-theme.ts`（或等价实现）
- `mactfsui/app/components/explorer/diff-dialog.tsx`、`file-view-dialog.tsx`（Monaco 主题接线）

## 验收标准

- 系统切换外观时应用自动跟随；手动选择优先于系统并可持久化
- 切换有平滑过渡，无闪烁
- dark 模式下所有界面可读性达标，无「白块」遗漏

```bash
# 手工验证：切换 macOS 系统外观 + 顶栏手动三态切换，全页面走查
```
