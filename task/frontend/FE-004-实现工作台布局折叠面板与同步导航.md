## 目标

实现新的工作台外壳，让顶部、左侧、中间、右侧、底部的职责清晰，并让左侧树和中间列表共享路径状态。

## 实现范围

- 顶部上下文栏
- 左侧 Source Tree 面板与收起状态
- 中间主工作区骨架
- 右侧 Changes 面板与收起状态
- 底部 Operation Console 与收起状态
- `home.tsx` 统一维护当前路径与同步导航状态

## 不在范围

- 不实现具体目录树数据加载
- 不实现文件对象右键菜单
- 不实现 History、Diff、目录对比弹窗
- 不做拖拽面板宽度

### 实际实现内容

- 顶部上下文栏只保留连接上下文（服务器 / Collection / Workspace）、面板开关与重新连接
- 左侧（280px）、右侧（340px）、底部（180px）面板均可展开 / 收起，开关位于顶部栏，图标 + tooltip
- 当前路径状态由 `home.tsx` 统一维护，左树点击根节点与中间列表共享同一路径状态
- 中间区域只保留当前路径、Mapping 摘要、列表占位、选中项摘要，未内嵌任何弹窗类内容
- 目录树数据加载、右键菜单、各业务弹窗均未实现（按任务边界留给 FE-005+）
