## 目标

实现固定 Collection 下的服务端目录浏览，并让左侧树和中间当前目录列表保持一致。

## 实现范围

- 左侧目录树懒加载
- 中间目录文件列表加载当前路径下一级内容
- 列表展示名称、状态、版本、本地路径、时间等核心列
- 已映射 / 未映射 / 未下载状态展示
- 单击选中、双击进入目录
- 左侧树选择目录与中间列表导航同步

## 不在范围

- 不实现右键菜单
- 不实现 Mapping、History、目录对比弹窗
- 不实现 Diff 展示
- 不实现 Pending Changes 面板

- 可浏览当前 Collection 下的服务端目录树
- 左侧目录切换时中间列表同步刷新

### 实际实现内容

- 左侧树固定 Collection，懒加载 `/api/server-tree`，仅展示 folder 节点，加载中有 spinner，空目录视为叶子
- 中间列表加载 `/api/server-folder/items`，展示当前路径下一级内容与映射状态
- 未映射目录可正常浏览；已映射项推导本地路径，结合 Electron 本地存在性检测区分「已映射 / 未下载」
- 单击行选中（底部显示名称、服务端路径、状态、版本），双击目录进入
- 左树点击与中间双击共用 `home.tsx` 的 `selectedServerPath`，树会自动逐级加载并展开祖先链实现同步
