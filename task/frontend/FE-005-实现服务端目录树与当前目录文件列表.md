## 目标

实现固定 Collection 下的服务端目录浏览，并让左侧树和中间当前目录列表保持一致。

## 实现范围

- 左侧目录树懒加载
- 中间目录文件列表加载当前路径下一级内容
- 列表展示名称、状态、版本、本地路径、时间等核心列
- 已映射 / 未映射 / 未下载状态展示
- 单击选中、双击进入目录
- 左侧树选择目录与中间列表导航同步

## 不在范围

- 不实现右键菜单
- 不实现 Mapping、History、目录对比弹窗
- 不实现 Diff 展示
- 不实现 Pending Changes 面板

## 验收标准

- 可浏览当前 Collection 下的服务端目录树
- 左侧目录切换时中间列表同步刷新
- 中间双击目录时左侧树同步展开和选中
- 未映射目录仍可浏览
- 已映射未下载项状态清晰
